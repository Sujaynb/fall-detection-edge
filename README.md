\# Edge-Based Fall Detection System



\## Overview

Edge-based fall detection web application using computer vision and machine

learning to detect fall events in real time with low latency.



\## Tech Stack

\- Python

\- OpenCV

\- YOLOv8 (Fall Detection)

\- Flask / FastAPI

\- Edge-based inference



\## Note on Data \& Models

Training datasets and trained model weights are excluded from this repository

to protect academic and intellectual property.
